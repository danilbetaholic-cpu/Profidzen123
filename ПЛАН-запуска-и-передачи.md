Три задачи в одном плане:
1. Передать сайт без следов разработчика (анонимно).
2. Сделать приём заявок в Telegram + чат с клиентом.
3. Выгрузить на белорусский хостинг под регламент bePaid.

# ЭТАП 1. Чистая сборка без следов разработчика

Цель: отдать файлы, в которых нет вашего имени, почты и истории.

1. **Не передавайте GitHub-репозиторий Profidzen123.** В истории его коммитов
   навсегда зашиты ваши имя и email — это след, который не стирается.
2. Соберите чистый комплект ТОЛЬКО из рабочих файлов:
   - `index.html`, `admin.html`, `products.js`, `payment.js`, `worker.js`, `robots.txt`

3. Передайте архив владельцу через мессенджер (лучше с автоудалением).
4. После полного запуска удалите свой репозиторий:
   GitHub → Profidzen123 → Settings → Delete this repository.

> Если хотите отдать «под ключ» уже работающий сайт — заводите все аккаунты

# ЭТАП 2. Домен .by и белорусский хостинг (требование bePaid)

bePaid НЕ примет сайт на `github.io` — это бесплатный домен 3-го уровня
на бесплатном хостинге, прямо запрещено регламентом. Нужно:

1. **Купить домен в зоне .by** (регистрируется в БелГИЭ автоматически при покупке).
   Например: `ibrain.by`. Регистратор — hoster.by, active.by, domain.by.
2. **Заказать белорусский хостинг** (обычный shared-хостинг подойдёт):
   - hoster.by, flyhost.by, hostfly.by — все белорусские, ~15–30 BYN/мес.
3. Оформлять домен и хостинг **на ИП Савельеву** (паспорт/УНП) — это же лицо
   будет в договоре bePaid, данные должны совпадать.
4. Залить файлы сайта в корень хостинга через панель управления (File Manager)
   или по FTP. Структура простая — это статические файлы, ничего настраивать не нужно.
5. Проверить, что сайт открывается по `https://ibrain.by` (SSL обязателен —
   на белорусских хостингах включается бесплатно в пару кликов, Let's Encrypt).

### Подчистить внешние ссылки (bePaid не любит «ссылки на другие ресурсы»)
Сейчас сайт тянет из интернета: фото Apple, шрифты Google, иконки. Перед bePaid
лучше скачать их на свой хостинг, чтобы всё было «под единым доменным именем»:
- шрифты (Manrope, Unbounded) → скачать и положить в папку `/fonts`;
- фото товаров → скачать в `/img` и поменять ссылки в `products.js`;
- Three.js (если используется) → положить локально.
Это разовая работа, помогу с ней при переезде.

# ЭТАП 3. Telegram-бот для заявок + чат с клиентом

Бот и заявки живут в воркере на Cloudflare — в файлах сайта их нет,
поэтому к анонимности и передаче это отношения не имеет.

### 3.1. Создать бота (2 минуты)
1. В Telegram найдите **@BotFather** → команда `/newbot`.
2. Задайте имя и логин бота (например `ibrain_orders_bot`).
3. BotFather пришлёт **токен** вида `123456:ABC-DEF...` — сохраните.

### 3.2. Создать чат, куда падают заявки
Вариант А — **личные уведомления**: напишите боту @userinfobot — он пришлёт
ваш `chat_id` (число). Заявки будут приходить вам в личку.

Вариант Б — **общий чат «Заказы iBrain»** (удобно для команды/владельца):
1. Создайте группу, добавьте туда вашего бота.
2. Добавьте бота @userinfobot в группу — он покажет `chat_id` группы
   (отрицательное число вида `-100...`). Потом @userinfobot можно удалить.
3. Чтобы дать доступ владельцу/менеджеру — просто добавьте их в эту группу.

### 3.3. Подключить
В файле `worker.js` впишите:
```js
const TELEGRAM_BOT_TOKEN = "ваш токен";
const TELEGRAM_CHAT_ID   = "123456789";        // или "-100..." для группы
// можно несколько через запятую: "123, -100..."
```
Готово: каждый заказ с сайта приходит в Telegram — товар, конфигурация,
имя, телефон, email, адрес, сумма и время.

### 3.4. Живой чат с клиентом на сайте (по желанию)
Чтобы клиент мог писать прямо с сайта, добавим плавающую кнопку
«Написать в Telegram» → открывает диалог с менеджером. Сделаю на этапе доработки.

# ЭТАП 4. Подключение bePaid по регламенту

1. **Зарегистрировать магазин в Торговом реестре РБ** — бесплатно, через
   исполком или одно окно, нужен готовый домен. 1–5 дней.
2. На сайте уже есть всё, что требует bePaid (сделано):
   - реквизиты ИП (наименование, УНП, адрес, режим работы, орган регистрации);
   - раздел «Возврат и обмен»;
   - раздел «Безопасность платежей» (SSL, Verified by Visa, Mastercard SecureCode,
     Интернет-пароль БЕЛКАРТ);
   - логотипы платёжных систем (Visa, Mastercard, БЕЛКАРТ, ЕРИП).
   ⚠️ Осталось вписать реальные телефон и email ИП.
3. **Подать заявку на bepaid.by** → «Подключиться». Документы ИП Савельевой:
   свидетельство о госрегистрации, паспорт, реквизиты счёта, ссылка на сайт `ibrain.by`.
4. Получить **Shop ID** и **Secret Key**, вставить в `worker.js`
   (`SHOP_ID`, `SHOP_SECRET`), поставить `TEST_MODE = false`.
5. В `products.js` включить боевую оплату:
   ```js
   payment: { mode: "bepaid", bepaidTokenUrl: "https://адрес-воркера" }
   ```
6. Проверить тестовый и реальный платёж.
