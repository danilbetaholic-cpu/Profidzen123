# Запуск магазина на WordPress + WooCommerce (пошагово)

Весь путь занимает 1–2 часа. Программист не нужен.

## Шаг 1. Хостинг и домен

1. Зайдите на **hoster.by** (крупнейший хостинг РБ — подходит для Торгового реестра).
2. Закажите тариф «Хостинг для WordPress» (~15–25 BYN/мес) и домен в зоне **.by** (нужен для Торгового реестра и bePaid).
3. В панели хостинга нажмите «Установить WordPress» — hoster.by ставит его в один клик.
   Логин/пароль администратора сохраните.

## Шаг 2. Базовая настройка WordPress

1. Откройте `ваш-домен.by/wp-admin`, войдите.
2. Настройки → Общие: язык «Русский», название «iStore.by».
3. Плагины → Добавить новый → установите и активируйте:
   - **WooCommerce** (сам магазин)
   - **Astra** (Внешний вид → Темы → найти «Astra» → активировать) — быстрая бесплатная тема в стиле Apple-минимализма. В мастере Astra выберите готовый шаблон электроники.
4. Запустится мастер WooCommerce: страна — Беларусь, валюта — **BYN**, отрасль — электроника.

## Шаг 3. Импорт всех товаров (файл уже готов)

1. В админке: **Товары → Импорт**.
2. Загрузите файл **woocommerce-import.csv** (лежит рядом с этой инструкцией).
3. Поставьте галочку «Обновлять существующие товары», нажмите «Продолжить» —
   соответствие колонок определится автоматически → «Запустить импорт».
4. Через минуту появится весь каталог: 41 товар с фото, ценами, категориями
   (iPhone, Б/У iPhone, iPad, Mac, Watch, AirPods, ТВ и дом, Аксессуары)
   и вариациями **Цвет / Память / SIM-карта** — цена меняется автоматически при выборе памяти.

Примечания:
- Фото подтягиваются с сервера Apple по ссылкам. Со временем замените на свои
  (Товары → товар → Изображение).
- Цены правятся в два клика: Товары → Изменить → Вариации.

## Шаг 4. Оплата bePaid (официальный плагин)

1. Зарегистрируйте магазин в **Торговом реестре РБ** (бесплатно, через исполком, нужен домен).
2. Подайте заявку на **bepaid.by** (документы ИП: свидетельство, паспорт, расчётный счёт, ссылка на сайт). Договор — 2–5 дней.
3. У bePaid есть **готовый модуль для WooCommerce**: скачайте его в личном кабинете bePaid
   или по инструкции на docs.bepaid.by → «CMS-модули» → WooCommerce.
4. Плагины → Добавить → Загрузить плагин → активировать.
5. WooCommerce → Настройки → Платежи → bePaid → вставьте **Shop ID** и **Secret Key** из кабинета.
6. Готово: карты Visa/Mastercard/БЕЛКАРТ, ЕРИП и Apple Pay принимаются автоматически,
   заказы падают в WooCommerce → Заказы (+уведомления на почту).

## Шаг 5. Обязательные страницы (требование РБ и bePaid)

Создайте через Страницы → Добавить:
- «Оплата и доставка»
- «Возврат и обмен»
- «Контакты» — с реквизитами: ИП, УНП, адрес, режим работы
- В подвал сайта добавьте: ИП Ф.И.О., УНП, дата регистрации в Торговом реестре.

## Полезное (по желанию)

- **Рассрочка**: bePaid поддерживает карты рассрочки (Халва и др.) — включается в кабинете.
- **Telegram-уведомления о заказах**: плагин «WooCommerce Telegram Notifications».
- **SEO**: плагин Yoast SEO или Rank Math.
- **Б/У товары**: категория «Б/У iPhone» уже создана импортом — состояние указано в описании.
