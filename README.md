# iStore.by — сайт по продаже iPhone

## Файлы

| Файл | Что это |
|---|---|
| `index.html` | Сам сайт (3D-анимация, каталог, корзина, оформление заказа) |
| `admin.html` | Админ-панель — редактирование цен и товаров |
| `products.js` | Товары, цены и настройки (можно править вручную) |
| `payment.js` | Модуль оплаты (bePaid / WebPay / ЕРИП) |

Чтобы посмотреть сайт — просто откройте `index.html` в браузере.
Для работы в интернете загрузите все 4 файла на любой хостинг (hoster.by, Netlify, GitHub Pages и т.д.).

## Как менять цены

**Способ 1 — админ-панель:** откройте `admin.html`, пароль по умолчанию `admin123`
(смените его в `products.js`, поле `adminPassword`). Меняйте цены, жмите «Сохранить».
Чтобы изменения увидели **все** посетители — нажмите «Скачать products.js» и замените
этим файлом старый на хостинге.

**Способ 2 — вручную:** откройте `products.js` в блокноте и поменяйте числа в поле `price`.

## Подключение bePaid для белорусского ИП (пошагово)

bePaid принимает карты Visa / Mastercard / БЕЛКАРТ, ЕРИП, Apple Pay / Google Pay.
Ставка ~2,8%. Работает с ИП — это стандартный случай.

### Шаг 0. Подготовка (важно для ИП в РБ)
1. **Расчётный счёт** в любом банке РБ (если ещё нет).
2. **Регистрация интернет-магазина в Торговом реестре РБ** — обязательное
   требование для приёма онлайн-платежей. Делается бесплатно через
   исполком или одно окно, обычно 1–5 дней. Понадобится домен сайта.
3. На сайте должны быть: ИП с УНП, юридический адрес, режим работы,
   способы оплаты и доставки (добавьте в подвал сайта свои реальные данные —
   сейчас там заглушки).

### Шаг 1. Заявка в bePaid
- Оставьте заявку на https://bepaid.by → «Подключиться».
- Документы ИП: свидетельство о госрегистрации, паспорт, реквизиты счёта,
  ссылка на сайт.
- Рассмотрение и договор — обычно 2–5 рабочих дней.

После подключения в личном кабинете вы получите **ID магазина (Shop ID)**
и **секретный ключ (Secret Key)**.

### Шаг 2. Выберите режим в `products.js` → `payment.mode`

**`"link"` — самый простой (без программиста).**
В кабинете bePaid/WebPay создайте платёжную ссылку или кнопку оплаты,
вставьте её в поле `paymentLink`. Покупатель будет переходить на страницу оплаты.

**`"bepaid"` — полноценный виджет (нужен мини-бэкенд).**
Секретный ключ нельзя класть на сайт, поэтому нужен крошечный сервер,
который создаёт checkout-токен. Пример на Node.js:

```js
// server.js  (npm i express node-fetch)
const express = require("express");
const app = express();
app.use(express.json());

const SHOP_ID = "ВАШ_ID";
const SECRET = "ВАШ_СЕКРЕТНЫЙ_КЛЮЧ";

app.post("/api/bepaid-token", async (req, res) => {
  const r = await fetch("https://checkout.bepaid.by/ctp/api/checkouts", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Authorization": "Basic " + Buffer.from(SHOP_ID + ":" + SECRET).toString("base64")
    },
    body: JSON.stringify({
      checkout: {
        transaction_type: "payment",
        order: {
          amount: req.body.amount,        // в копейках
          currency: req.body.currency,
          description: req.body.description
        }
      }
    })
  });
  const data = await r.json();
  res.json({ token: data.checkout.token });
});

app.listen(3000);
```

Затем в `products.js` укажите:
```js
payment: { mode: "bepaid", bepaidTokenUrl: "https://ваш-домен/api/bepaid-token" }
```

**`"demo"` — режим по умолчанию.** Оплата имитируется — удобно для проверки сайта.

### ЕРИП
Подключается автоматически через bePaid или WebPay — отдельная интеграция не нужна,
покупатель просто выбирает ЕРИП на странице оплаты.

## Важно
- Смените пароль админки (`adminPassword` в `products.js`).
- 3D-анимация подгружает Three.js из интернета; без сети сайт работает, просто без 3D.
